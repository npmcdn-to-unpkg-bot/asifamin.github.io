Main advantages of migrating to Angular 2

  • Faster Plateform
  • Simpler to use
  • Forward looking
  • Supports Additional features like lazy loading, Server-Side Rendering Features

Change Detection

Events on Angular 1 (ex click event) also trigger Angular 2 change detection similarly events on Angular 2 (ex click event) also trigger Angular 1 digest cycle. Because of this migrating from Angular 1 to Angular 2 is straight forward

Basic overview of the migration plan

  1. Upgrade to the latest Version of Angular 1
  2. All new Development with Components
  3. Switch Controllers to Components
  4. Switch Component Directives to Components
  5. Implement Manual Bootstraping
  6. Development using TypeScript and ES6 (Optional)
    • Switch Controllers to ES6 Classes
    • Switch Services to ES6 Classes
